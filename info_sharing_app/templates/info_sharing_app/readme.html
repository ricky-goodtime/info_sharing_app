本ポートフォリオについて

  • Python の Django というフレームワークを使用して、情報共有ツールを作成しました。
  • 4つの基本機能(Create, Read, Update, Delete)に加えて、認証機能が組み込まれています。
  • サーバサイド開発に焦点を当てているため、HTML/CSSは非常にシンプルなコードになっております。個人で活動していた際にサンプルとして公開していたWebページ(サンプル1, サンプル2)があるため、必要に応じてご覧いただけますと幸いです。
  • GitHubへのリンク

機能

  • ホームディレクトリでは、作成された情報の一覧を閲覧することができます。タイトルが詳細画面へのリンクになっています。
  • 認証機能：画面右上のリンクからログインすることで、1. 情報の作成(シェア)、2. 情報の編集、3. 情報の削除、4. 情報へのコメント が可能になります。
  • フォームのバリデーション機能は Django に用意されているものを使用しています。
